# Product Quality Lessons

> product-quality 에이전트가 테스트 시나리오를 작성할 때 반드시 참조하는 파일입니다.
> 사용자가 발견한 버그 = 이 에이전트가 놓친 시나리오.

## 항상 포함할 리그레션 시나리오

(누적 — 매 테스트 계획에 반드시 포함)

1. 독 아이콘 호버 시 줌이 60fps로 부드럽게 동작
2. 독 자동 숨김이 윈도우 겹침 시 정상 작동
3. 독의 기존 기능(줌, 클릭, 드래그)이 새 기능 추가 후에도 정상 작동
