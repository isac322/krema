# Krema Positioning & Messaging

## One-Liner

> A lightweight, high-performance dock for KDE Plasma 6 — spiritual successor to Latte Dock.

## Elevator Pitch

Krema is the dock that KDE Plasma 6 users have been waiting for since Latte Dock
was discontinued. Built from scratch with C++23 and KDE Frameworks 6, it delivers
smooth parabolic zoom animations, live PipeWire window previews, and deep native
integration with the Plasma desktop. It's not a generic Linux dock — it's designed
specifically for Plasma, using Kirigami and KDE color schemes for a truly native
experience.

## Value Propositions

### For Former Latte Dock Users
**"Your dock is back."**
- Krema picks up where Latte Dock left off, purpose-built for Plasma 6
- Familiar parabolic zoom, drag-and-drop, smart visibility modes
- Kirigami-based settings dialog for easy customization

### For KDE Plasma Enthusiasts
**"A dock that speaks Plasma."**
- Uses KDE's own frameworks: Kirigami, KConfig, KColorScheme, KGlobalAccel
- Respects your Plasma theme colors, global shortcuts, and desktop conventions
- Window previews powered by PipeWire — the same stack Plasma uses internally

### For Performance-Conscious Users
**"Lightweight by design."**
- GPU-accelerated rendering via QRhi (Vulkan/OpenGL)
- Wayland-native via Layer Shell — no X11 compatibility hacks
- Lazy resource allocation; GPU resources released when dock is hidden

### For Customizers (r/unixporn)
**"Make it yours."**
- Multiple background styles: transparent, tinted, acrylic frosted glass
- Configurable icon size, zoom factor, spacing, dock position
- Integrates seamlessly into any Plasma rice

1. **Only actively maintained dock with parabolic zoom for Plasma 6**
2. **Wayland-native from day one** — no X11 legacy code
3. **Deep KDE integration** — not just running on KDE, built WITH KDE libraries
4. **PipeWire-based window previews** — modern, efficient, Plasma-native
